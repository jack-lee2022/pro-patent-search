# Pro Patent Search Skill

An expert-level Agent Skill for IP researchers, transforming AI agents into **Virtual Patent Engineers**.

## SOP for Agents
This skill provides a professional workflow for:
1. **Invalidity Search**: Priority date strict filtering.
2. **FTO Analysis**: Claim Construction and Element Mapping.
3. **Patent Landscape**: Citation Snowballing and Assignee Trend Mapping.

## Integration
- Built upon `patent-search-engine` core scripts.
- Compatible with: Gemini CLI, Claude Code, OpenClaw, Hermes Agent.

## Usage
Trigger with: "Perform an FTO analysis for [Technology]", "Generate a Claim Chart for patent [ID]".
See `SKILL.md` for full SOP.
